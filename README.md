# GEMUG

기술적으로 준비된 심사·컨설팅형 개인정보보호 담당자를 지향합니다.

## About

- **관심 분야**: Privacy Compliance · Data Governance · Technical Privacy · ISMS-P
- **목표 포지션**: 개인정보보호 법령을 기술 요구사항으로 번역하고, 실제 데이터 구조와 시스템을 점검해 개선안을 도출하는 기술형 개인정보보호 담당자
- **배우는 중**: CPPG · SQL/DB · 정규표현식 기반 개인정보 탐색 · GitHub 기반 증적화
- **자격증 로드맵**: CPPG → 정보처리기사 → SQLD → 정보보안기사 → AWS CLF/SAA → PIA → ISMS-P 인증심사원 → CISA → CISSP
